# Higgsfield Image & Video Workflow

For generating South Park-style images and video clips via Higgsfield,
follow the workflow in `workflows/higgsfield-south-park.md`.

**Requires Playwright MCP** — install with:
```bash
claude mcp add playwright npx '@playwright/mcp@latest'
```

## Default Higgsfield Settings
- Tool: NanoBanana Pro (images), Cinema Studio (video)
- Aspect ratio: 16:9
- Image count: 8
- 2K unlimited: ON
